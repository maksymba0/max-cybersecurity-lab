# Dota 2 Network Traffic Analysis

**Start Date:** March 2026  
**Tools:** Wireshark, Dota 2

**Objective:**  
Analyze game traffic to understand network behavior and practice security analysis.

**Results:**
- Identified Valve server in Warsaw (155.133.230.98)
- Confirmed UDP protocol (100%), encryption in use
- Isolated chat packets by size (>400 bytes)

**Why This Matters:**  
Learning to identify normal vs abnormal traffic patterns.
