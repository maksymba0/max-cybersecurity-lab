Date: 20/03/2026
Tools: Dota 2, Wireshark,  whatismyipaddress.com
Objective: Analyze and investigate Dota 2's network traffic. 

Tasks:

1. Starting a private lobby in Dota 2 with AI bots; 

2. Launching Wireshark to start capturing packets going in & out. 

3. Performing different network in-game activities (sending chat message, hero movement, using spells) for a few minutes.

4. Stopping capture of packets.

5. Filtering the packets and manual investigation.

6. Obtaining the IP address which belongs to Valve  and used for communicating with client's side

7. Used IP lookup to confirm the server belongs to Valve (whatismyipaddress.com)


Results:

1. Valve and Dota's network system uses UDP protocols (not port 27015)

2. Valve's Data Server in Poland is located in Warsaw.

3. The Valve's Data IP address: 155.133.230.100

4. Attached screenshots as proof. 


Purpose:
This helps to investigate where the data is coming from and going to, helps verify if the server data being sent to is valid source or being abnormal (cybersecurity basis).
